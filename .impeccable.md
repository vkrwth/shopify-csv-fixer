## Design Context

### Users
Shopify store owners or staff responsible for product CSV imports. Doing a job, not exploring. They have a broken file, they want it fixed. Impatience is the default state.

### Brand Personality
Fast, no-nonsense, practical. The interface should feel like a sharp tool — no ceremony, no fluff. Every element earns its place by doing something useful.

### Aesthetic Direction
Warm document-minimalism. Off-white canvas (#FBFBFA), near-black ink (#111111), hairline borders, muted pastel badges. Light mode only. Instrument Serif for display headings (editorial contrast), DM Sans for body (clean, readable at small sizes), JetBrains Mono for labels and codes. No gradients, no shadows beyond 4px diffuse, no decorative elements.

### Design Principles
1. **Clarity over cleverness** — if someone has to think about what to do next, the design failed.
2. **Trust through precision** — exact labels, exact numbers, no vague copy. The user is about to import products into a live store.
3. **Speed signals respect** — instant feedback, no loading states that linger, no animation that delays the user.
4. **Earn every pixel** — if an element doesn't help the user complete the task, remove it.
5. **Quiet confidence** — the tool should feel obviously correct, not try-hard polished.
